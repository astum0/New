    Что такое GitHub?

    GitHub - это веб-интерфейс, использующий Git, программное обеспечение с открытым кодом для контроля версий, которое позволяет
    нескольким людям одновременно вносить отдельные изменения на веб-страницы.

    Для чего используют GitHub?

    GitHub используют например для хранения
    веб-сайтов или программ.
    В GitHub размещается основной репозиторий
    всех проектов. Из этого репозитория участники копируют свою работу.

    Для чего используется Git Bash?

    Git Bash - это командная строка в Windows, которая позволяет выполнять команды Git. Помимо них, она содержит все стандартные команды Bash. Git GUI - это графический интерфейс для Git, который позволяет выполнять, все основные операции с Git без использования командной строки.
